# Tech Stack

## Frontend
*   **Framework:** React
*   **Language:** TypeScript
*   **Build Tools:** Webpack, Babel
*   **Linting:** ESLint
*   **HTTP Client:** Axios
*   **Styling:** Tailwind CSS (for microfrontend-one)

## Backend
*   **Runtime:** Node.js
*   **Framework:** Express.js
*   **Language:** TypeScript
*   **Development Tools:** ts-node, dotenv

## Build & Package Management
*   **Package Manager:** pnpm

## Architecture
*   **Pattern:** Monorepo with client-server and microfrontend patterns
